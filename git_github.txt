Git- это распределенная система контроля  версий,предназначен для отслеживания и введения истории измененич файлов,в вашем проекте

Github- это сайт куда мы можем отправлять все что находиться на нашем ноутбуке,чтобы другие люди могли посмотреть на мой  проект и стянуть(скачать ) к себе
AIDAI KYDYKOVA 

1.git --init (чтобы создать скрытую директорию (.git )), инициаоизация 
2. git add .-  просмаривает изменения в нашем проекте (если хотим чтобы просматривал определенный то 
полс еточки пишем название файла )и добавляет в промежуточное проста=ранство 
3.git commit -m 'message' -фиксирует изменения из индекса в локальный репазиторий 
4.git remote add (название связки  (origin))- связывает локальный и удаленный репазиторий 
5.git push (название связки(origin) main либо master )- отправляет данные из локального в удаленный репазиторий 
6.git pull origin master (main)- отправляет данные из удаленного в локальный репазиторий 

закидывваем изменения  в готовый репазиторий
1.git add git_github.txt(название файла)
2.git commit -m 'update file git_github.txt'
3.git push origin master(main)
 
Стягиваем изменения из удаленного репазитория
1. git clone (ссылка)
2.git pull origin master
