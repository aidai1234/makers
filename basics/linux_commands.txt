ls-показывает содержимое папки
ls  название директории - показывает все файлы и папки внутри папки  
ls-a =показывает вскрытые файлы и папки 
ls-l= подробная информация в файлах и в папке 
ls-la =показыввает дополнительную информацию и скрытые файлы с подробной инфо
cd-переход в домашнюю траекторию
cd~=перехрод в домашнюю траекторию 
cd test= переход в папку тест  если такой диреткории нет  то выйдет ошибка
cd..=переход на директорую выше 
cd../..=переход нв две директории выше 
cd-=cntr -z = выполняет предыдущую команду 
mkdir test =создает папку тест в текущей директории 
mkdir test/test2= создает папку тест 2 внутри папки тест если папка тест существует
mkdir test test2=-создает папку тест и тест 2 в текущей директории 
touch file.txt= создает файл с разрешением txt 
touch file1.txt file2.txt = создает файл 1 и файл 2 с разрешение txt 
rm file1 file2.txt =удаление файла 
rmdir test1 test2 =удаление папки пустой 
rm-rf test  = удаляет все и файлы и папки содержимые в них ( полное удаление) 
mv test.txt test = перемещает файл test.txt в директорию test 
mv test1.txt test2.txt =переименовываает файл test1.txt на test2.txt
pwd =показывает абсолютный  путь (начиная с корневой папки /) до текущей директории  
history=история команд  можно сделать лимит к примеру history 5,history20
history|grep test = показывает все команды которые делались до этого со словом test   
nano test.txt = открывает файл test.txt в текстовом редакторе nano 
cat test.txt t= выводит содержимое файла  в терминале
sudo =права суперюзера 
apt= системный  установщик пакетов
brew = установщик пакетов ( для маков)
